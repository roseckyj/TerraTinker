# Comment node

Comment node is a [passive node](/layers/node_types/passive) that does nothing. It is used to add comments to the graph.

It can be modified by clicking on it.

    {
        "type": "comment",
        "location": [0, 0],
        "inputs": {},
        "nodeData": {
            "content": "This is a comment"
        }
    }
</Node>
