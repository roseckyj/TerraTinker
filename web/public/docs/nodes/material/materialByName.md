# Material By Name node

The Material By Name node allows you to select a material by its name.

<Node>
    {
        "type": "materialByName",
        "location": [0, 0],
        "inputs": {},
        "nodeData": {}
    }
</Node>

## Inputs

-   `Material Name`: Name of the material to select.

## Outputs

-   `Material`: Found material.
